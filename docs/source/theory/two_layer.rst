Two-Layer Planar Geometry
=========================

.. note::

   This section is under development. It will cover the derivation of the dyadic
   Green's function for a planar two-layer geometry (e.g. vacuum above a metallic
   substrate) using Sommerfeld integrals, including the decomposition into TE/TM
   modes and the Fresnel reflection coefficients that enter the scattering
   Green's function :math:`\overline{\overline{\mathbf{G}}}_\mathrm{Sc}`.
